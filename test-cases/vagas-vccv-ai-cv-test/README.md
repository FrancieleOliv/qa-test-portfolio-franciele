# AI Resume Builder – Vagas.com

This folder contains all documentation and evidence for QA testing of the AI-powered resume builder from Vagas.com.

## Test Documentation

- [Test Report](./test-report.md) – Summary of the testing process, scenarios, and results.
- [Bug Report](./bug-report.md) – Detailed description of identified defects.
- [Evidences](./evidences/) – Screenshots and step-by-step proof for each bug.

## Tested Scenarios

- Upload resume as PDF → System should automatically extract resume data.
- Upload resume as image → System should recognize all text content.
- Manual text insertion → System should optimize and structure the content.
- Interaction with the tool's chatbot → System should maintain consistent language and instructions.

## 🇧🇷 Português

# Criador de Currículos com IA – Vagas.com

Esta pasta contém toda a documentação e evidências dos testes de QA realizados na ferramenta Criador de Currículos com IA da Vagas.com.

## Documentação de Teste

- [Relatório de Teste](./test-report.md) – Resumo do processo de teste, cenários e resultados.
- [Relatório de Erros](./bug-report.md) – Descrição detalhada dos defeitos identificados.
- [Evidências](./evidences/) – Capturas de tela e provas passo a passo de cada bug.

## Cenários Testados

- Upload de currículo em PDF → O sistema deve extrair automaticamente os dados do PDF.
- Upload de currículo como imagem → O sistema deve reconhecer todo o conteúdo textual.
- Inserção manual de texto → O sistema deve otimizar e estruturar o conteúdo.
- Interação com o chatbot da ferramenta → O sistema deve manter consistência no idioma e instruções.
