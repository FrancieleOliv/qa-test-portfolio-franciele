## Evidence

This folder contains test evidence organized by scenario.

Each subfolder represents a different test type:

- **pdf-upload** → PDF upload test evidence
- **image-upload** → Image upload test evidence  
- **text-input** → Manual text input test evidence  

Each folder includes screenshots demonstrating the system behavior and identified issues.
